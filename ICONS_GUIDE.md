# Icons8 Line Awesome - Icon Mapping Guide

## What Changed?

We've replaced Font Awesome with **Icons8 Line Awesome** - a free, open-source icon library.

## Icon Class Changes

Simply replace `fa` or `fas` or `fab` with `la` or `lab`:

### Before (Font Awesome):
```html
<i class="fas fa-home"></i>
<i class="fab fa-github"></i>
```

### After (Icons8 Line Awesome):
```html
<i class="las la-home"></i>
<i class="lab la-github"></i>
```

## Icon Styles

Icons8 Line Awesome has different styles:

- `las` - Line Awesome Solid (like Font Awesome's `fas`)
- `lar` - Line Awesome Regular (like Font Awesome's `far`)
- `lab` - Line Awesome Brands (like Font Awesome's `fab`)
- `lal` - Line Awesome Light

## Example Usage in Components

```tsx
// Home icon
<i className="las la-home"></i>

// GitHub brand icon
<i className="lab la-github"></i>

// Regular copyright icon
<i className="lar la-copyright"></i>
```
